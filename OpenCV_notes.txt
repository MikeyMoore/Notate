1. Install Xcode
2. Install CMake: www.cmake.org
3. Download OpenCV source code
4. Create new folder inside opencv folder, name it 'build'
5. Open CMake (it might throw a warning since it'd from an unidentified developer source)
6. On the source code, browse source and set it to the OpenCV folder you just downloaded.
7. On the Where to build binaries, browse build and set it to the build folder you just created.
8. Hit configure, specify Unix Makefiles and click button for Use default native compilers.
9. Click Generate.
10. In terminal, cd to the build folder you created.
11. Type make
12. Type sudo make install
